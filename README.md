#Video Action Classification

 **about project:**

 **dataset_links**
  https://www.deepmind.com/open-source/kinetics
  https://www.tensorflow.org/datasets/catalog/ucf101
  
  We used the 2 datasets to make a prediction but used ucf101 for production.
